# Eisenhower Matrix Task Reminder

A smart, single-file task management tool based on the Eisenhower Matrix method. It helps you prioritize tasks by urgency and importance.

## Features

- **Eisenhower Matrix Visualization**: 4 quadrants to organize tasks (Do Now, Schedule, Delegate, Delete).
- **Smart Reminders**: Browser notifications trigger exactly 15 minutes before a task is due.
- **Local Persistence**: Tasks are saved in your browser's local storageâ€”no data is lost on refresh.
- **Premium UI**: Clean, modern interface with pastel colors and smooth interactions.

## How to Use

1. **Add a Task**: Enter a title, select a deadline, and choose the appropriate quadrant.
2. **View Context**: See your tasks visually organized to know what to focus on.
3. **Get Notified**: Keep the tab open to receive notifications 15 minutes before deadlines.

## Setup

Simply open `index.html` in any modern web browser. No installation or server required.
